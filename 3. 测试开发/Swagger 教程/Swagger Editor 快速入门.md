# 简介

> Swagger 是最受欢迎的REST APIs文档生成工具之一
>
> Swagger 可以生成一个具有互动性的API控制台，开发者可以用来快速学习和尝试API。
>
> Swagger 可以生成客户端SDK代码用于各种不同的平台上的实现。
>
> Swagger 文件可以在许多不同的平台上如node，java从代码注释中自动生成。
