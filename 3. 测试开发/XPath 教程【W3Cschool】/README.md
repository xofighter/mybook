# Introduction

## XPath 教程

XPath 是 XML 路径语言。

XPath 是一门在 XML 文档中查找信息的语言。

XPath 是 XSLT 中的主要元素。

XQuery 和 XPointer 均构建于 XPath 表达式之上。

**现在开始学习 XPath !**
